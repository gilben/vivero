# pj6-frontend

Generated by **lv-agents** — React 18 + Vite + TypeScript.

## Quick start

```bash
npm install
cp .env.example .env
npm run dev
```

- **Claude Projects**: paste `CLAUDE.md` into Project Instructions

Add `.md` files to `docs/user-stories/` and run:

npx agent-docs
